Dashboard Data Analyst
Project ini dibuat sebagai bagian dari technical test seleksi Data Analyst pada program Dispora Jawa Barat.

Deskripsi Project
Dashboard ini dibuat menggunakan Power BI untuk menganalisis data dan menampilkan insight dalam bentuk visualisasi data interaktif.

Tools yang Digunakan
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization

Fitur Dashboard
- Analisis penjualan
- Analisis pelanggan
- Tren pendapatan
- Visualisasi dashboard interaktif
